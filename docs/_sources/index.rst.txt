MoHa |version|
##############
MoHa is abbreviation of *Mo*\lecular/*Mo*\del *Ha*\miltonian, it is a quantum chemistry program written in python. MoHa now is not a formal program but a toy for practice coding and help myself to have a better
understanding of quantum chemistry. If you are interested in, Please see more details in the :ref:`Overview`.
