Installation For Linux and macOS
************
* Prerequisties:

  - Python 3.5 or above
  - numpy 1.13.1 
  - scipy 0.19.1  

* Download moha

  - gh repo clone ZhaoYilin/FanCI

* Compile

  - cd moha
  - python3 -m pip install -e .

* Run the following to test moha:

  - pytest 

* Build documentation

  - cd docs
  - make html
